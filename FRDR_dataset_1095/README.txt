Description of methods used for collection/generation of data: The methodology is described fully in Baulch et al. (2025). A summary is provided below.Data were collected using a sensor array for weather and water quality-related variables on a high-frequency monitoring buoy. Data were collected at subhourly timescales during the ice-free period. 2. Methods for processing the data: Methodology for processing data are described fully in Baulch et al. (2025). A summary is provided below.Data were compiled from raw data files into Excel spreadsheets. Data were initially reviewed and flagged for all known errors, including sensor or system failures, through an automated screening system to help grade data quality followed by detailed visual inspection. Corrections applied to 2017 specific conductivity data as described in Baulch et al. (2025).3. Instrument- or software-specific information needed to interpret the data: Not required.4. Standards and calibration information, if appropriate: Sensors were calibrated prior to deployment following device operating procedures as discussed in Baulch et al. (2025).5. Environmental/experimental conditions: Extreme winds and sensor fouling occasionally compromised instrumentation causing data gaps over the monitoring period.6. Describe any quality-assurance procedures performed on the data: Described in Baulch et al. (2025).-----------------------------------------DATA-SPECIFIC INFORMATION FOR: BPBuoyData_[2014/2015/2016/2017/2018/2019/2020/2021]_Cleaned.csv            -----------------------------------------1.
